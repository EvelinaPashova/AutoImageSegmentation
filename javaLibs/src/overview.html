This document is the API specification for the PixelMaster application. It is intended only for internal use (by the application's developers).

PixelMaster is an image processing application developed in {@link The Institute of Mathematics and Computer Sciences, University of Latvia (IMCS UL)}. PixelMaster is intended as a platform for implementing various image processing algorithms.
